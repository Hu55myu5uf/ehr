EHR System — Manual Guide

Setup Instructions & Testing Walkthrough

Part A: Application Setup & Startup

Step 1 — Start XAMPP Services

  1. Open XAMPP Control Panel
  2. Click Start next to Apache
  3. Click Start next to MySQL
  4. Both should show green "Running" status

Step 2 — Run Database Migrations

Open PowerShell and run:

Get-Content c:\xampp\htdocs\ehr\database\schema.sql | c:\xampp\mysql\bin\mysql.exe -u root

Get-Content c:\xampp\htdocs\ehr\database\migration_clinical_workflow.sql | c:\xampp\mysql\bin\mysql.exe -u root

Step 3 — Install Backend Dependencies

cd c:\xampp\htdocs\ehr\backend
composer install

Step 4 — Configure Backend Environment

Ensure c:\xampp\htdocs\ehr\backend\.env contains:

DB_HOST=localhost
DB_NAME=ehrecords
DB_USER=root
DB_PASS=
JWT_SECRET=<your-secret-key>
APP_DEBUG=true

Step 5 — Install Frontend Dependencies

cd c:\xampp\htdocs\ehr\frontend
npm install

Step 6 — Start the Frontend Dev Server

cd c:\xampp\htdocs\ehr\frontend
npm run dev

Step 7 — Open the Application

  1. Open your browser and navigate to http://localhost:5173
  2. Login as admin@ehr with your admin password

Services Reference

ServiceURLHow to Start
MySQLlocalhost:3306XAMPP → Start MySQL
Apache (Backend API)localhost/ehr/backend/public/XAMPP → Start Apache
Frontendlocalhost:5173npm run dev in frontend/
Note: Apache serves the PHP backend API via XAMPP. The frontend Vite dev server proxies API calls to it. Both must be running simultaneously.

Part B: Manual Testing Guide

Phase 1 — Super Admin Login & User Creation

1.1 Login as Super Admin

  1. Navigate to /login
  2. Enter username: admin@ehr, password: your admin password
  3. Complete MFA if prompted
  4. ✅ Verify: Dashboard shows "Admin Dashboard" title with all 6 KPI cards

1.2 Create User Accounts

Go to User Management (/admin/users) and create:

UsernameRolePassword
doctor1Doctor(your choice)
nurse1Nurse(your choice)
receptionist1Receptionist(your choice)
labtech1Lab Attendant(your choice)
pharmacist1Pharmacist(your choice)

✅ Verify: All 5 users appear in the user list

Phase 2 — Receptionist Workflow

2.1 Login as Receptionist

  1. Log out, log in as receptionist1
  2. ✅ Verify: Sidebar shows only: Dashboard, Patients, Appointments, Billing
  3. ✅ Verify: Dashboard shows "Reception Desk" title

2.2 Register a New Patient

  1. Go to Patients → click Register Patient
  2. Fill in: first name, last name, DOB, gender, phone, next of kin details
  3. Submit the form
  4. ✅ Verify: Patient appears with auto-generated MRN

2.3 Book an Appointment

  1. Go to Appointments → click Book Appointment
  2. Search and select the patient, set today's date, choose time and type
  3. Add a reason (e.g. "Fever and headache"), click Book Appointment
  4. ✅ Verify: Appointment appears with status "Scheduled"

2.4 Check In the Patient

  1. Hover over the appointment → click Check In
  2. ✅ Verify: Status changes to "Checked In"

Phase 3 — Doctor Workflow

3.1 Login as Doctor

  1. Log out, log in as doctor1
  2. ✅ Verify: Sidebar shows: Dashboard, Patients, Appointments, Consultations, Laboratory, ICU

3.2 Start a Consultation

  1. Go to Encounters → click Start Consultation
  2. Search for the patient, fill in type and chief complaint, submit
  3. ✅ Verify: Encounter appears in the list

3.3 Fill the Consultation Form

  1. Go to Consultations → click the encounter
  2. Fill each section:
    • Patient's Complaint — chief complaint and HPI
    • Review of Systems — check relevant organ systems
    • Past Medical / Drug / Family / Social History
    • Diagnosis — primary diagnosis + ICD code (e.g. "Malaria", "B54")
    • Treatment Plan — admission decision, nursing instructions
  3. Click Save
  4. ✅ Verify: "Saved successfully" message appears

3.4 Order Lab Tests (inline)

  1. In the Lab Orders section, type test name (e.g. "Full Blood Count")
  2. Select priority, click Order
  3. ✅ Verify: Test appears with status "ordered"

3.5 Prescribe Medications (inline)

  1. In the Medications section, enter name, dosage, frequency
  2. Select route, click Prescribe
  3. ✅ Verify: Medication appears with status "pending"

3.6 Complete Consultation

  1. Click Complete → confirm the dialog
  2. ✅ Verify: Encounter shows "Completed"; bill is auto-generated

Phase 4 — Nurse Workflow

  1. Log out, log in as nurse1
  2. ✅ Verify: Sidebar shows: Dashboard, Patients, Nursing, ICU
  3. Go to Nursing → select the patient
  4. ✅ Verify: Doctor's nursing instructions appear
  5. Click Record Vitals → fill in Temperature, BP, HR, SpO₂, etc. → Save
  6. Click Add Note → enter care note → Save
  7. ✅ Verify: Both appear in the timeline

Phase 5 — Lab Technician Workflow

  1. Log out, log in as labtech1
  2. ✅ Verify: Sidebar shows: Dashboard, Laboratory
  3. Go to Laboratory → find the pending order
  4. Update status to "In Progress", add results, mark "Completed"
  5. ✅ Verify: Result values display correctly

Phase 6 — Pharmacist Workflow

  1. Log out, log in as pharmacist1
  2. ✅ Verify: Sidebar shows: Dashboard, Pharmacy
  3. Go to Pharmacy → find pending prescriptions
  4. Click Dispense on each medication
  5. ✅ Verify: Status changes to "Dispensed"

Phase 7 — Billing & Payment (Receptionist)

  1. Log out, log in as receptionist1
  2. Go to Billing → click the pending bill
  3. ✅ Verify: Itemized list shows consultation fee (₦5,000), lab tests (₦2,500 each), medications (₦1,500 each)
  4. Click Process Payment → enter amount, select method → Confirm
  5. ✅ Verify: Bill status changes to "Paid"

Phase 8 — Super Admin Verification

  1. Log in as admin@ehr
  2. ✅ Verify: All 11 sidebar items visible
  3. ✅ Verify: Can access every module
  4. Go to User Management → try suspending and reactivating a user
